# Team 1: District-level zero-dose burden

Branch: `family-zero-dose`

Use this workspace for the team’s manuscript, approved code, data documentation, maps, tables, presentation and final prototype files.

## Recommended folders

- `manuscript/` — manuscript-style report covering background, methods, results, discussion, limitations and recommendations
- `code/` — data preparation, estimation, mapping and validation scripts
- `data/` — data dictionary, access instructions and approved public/de-identified data only
- `documentation/` — methods, assumptions, limitations and user guidance
- `outputs/` — final tables, maps, figures or application files
- `slides/` — final team presentation

## README requirements

Document the decision question, intended users, data sources, estimation method, geographic level, run order, outputs, validation, uncertainty, limitations and next step.

When complete, open a pull request to `hackathon-workspaces`.

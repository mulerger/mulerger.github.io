# Family 4: Measles transmission modelling

Branch: `family-measles-transmission`

Use this workspace for the family’s approved model code, parameters, assumptions, scenarios, data documentation, outputs, presentation and final prototype files.

## Recommended folders

- `code/` — model, calibration, scenario and sensitivity scripts
- `data/` — data dictionary, access instructions and approved public/de-identified data only
- `documentation/` — equations, parameters, assumptions, limitations and user guidance
- `outputs/` — model results, scenarios, figures or application files
- `slides/` — final family presentation

## README requirements

Document the policy question, intended users, model structure, parameters, data sources, calibration, validation, scenarios, run order, outputs, uncertainty, limitations and next step.

## Data rule

Do not upload identifiable case-level or restricted surveillance data. Use approved secure storage and provide reproducible scripts plus a data-access note.

When complete, open a pull request to `hackathon-workspaces`.

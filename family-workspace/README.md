# Family 4: Measles transmission modelling

Use this workspace for the family’s manuscript, approved model code, parameters, assumptions, scenarios, data documentation, prototype, dashboard if developed, outputs and presentation.

## Recommended folders

- `manuscript/` — manuscript-style report covering background, methods, results, discussion, limitations and recommendations
- `code/` — model, calibration, scenario and sensitivity scripts
- `data/` — data dictionary, access instructions and approved public/de-identified data only
- `documentation/` — equations, parameters, assumptions, limitations and user guidance
- `prototype/` — runnable final prototype and setup instructions
- `dashboard/` — dashboard source and deployment notes, if developed
- `outputs/` — model results, scenarios and figures
- `slides/` — final family presentation
